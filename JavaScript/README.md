# Gilded Rose in Javascript with Jest

## Getting started

Ensure you have NodeJS installed

Install node_modules dependencies from the command-line

```sh
npm install
```

## Run the unit tests from the Command-Line

To run all tests

```sh
npm test
```

To run all tests in watch mode

```sh
npm run test:watch
```

To generate test coverage report

```sh
npm run test:coverage
```

## Run the example simulation from the Command-Line

For e.g. 10 days:

```
node test/example_simulation.js 10
```
